# 숫자 비교하기 게임

## 두개의 팀이 5번에 걸쳐서 랜덤으로 나오는 숫자의 합으로 승패를 결정

<br/>

## 스킬
1. html
2. css3
3. javascript
4. react

## 기능적 정리
1. 1~9까지의 숫자를 랜덤으로 뽑기
2. 한 게임당 5번의 숫자를 뽑을 수 있음
3. 뽑은 5개의 숫자의 합을 구함
4. 5번의 숫자를 다 뽑으면 총 합이 구해지고 A팀, B팀의 승패가 결정됨
